REplaceOlder

A jQuery plugin for placeholder in older browsers

How to use?

  1. In $(document).ready(function(){}); section;
  2. Put $('input, textarea').placeholder(); code;
  3. Change the color of placeholder with .placeholder { color: #999; } in your CSS;
  4. Done!

Sample:

Features:

Very easy to use! Too light to load (only 6 KB)!

Work in all browsers!

Install:


		<script src="http://code.jquery.com/jquery-latest.js"></script>
		<script src="jquery.replaceOlder.js"></script>

The code:


		<script type="text/javascript">
			$(document).ready(function() {
				$('input, textarea').rOlder();
			});
		</script>
	 	<style type="text/css">
			.placeholder { color: #999; }
	 	</style>
